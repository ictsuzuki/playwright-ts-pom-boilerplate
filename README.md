This project was created for a case study..

## Getting started

### Prerequisites
Ensure you have the following prerequisites installed:

- [Node.js](https://nodejs.org/en/download/) package manager
- Any Code editor such as Visual studio code, Sublime, etc. 

### Installation
1.- Clone the github **repository** to the desired folder
```
git clone https://github.com/ictsuzuki/deel_interview.git
```
2.- Run your code editor and open the project folder

3.- Open a **New Terminal**

4.- Navigate to ```deel_interview``` as project root

5.- Create a ```.env``` file

6.- Add the next variable to the ```.env``` file. The value will be provided on the email. 

    ```BASE_URL= [get this value fomr the email] ```

7.- Run the next command to install all the dependencies

    npm install

8.- Install the playwright browsers

    npx playwright 

9.- Run any of the next commands on the terminal to run the 
tests:

    npm run test_chrome 
    npm run test_all_browsers


Please feel free to provide feedback and improvements.

Happy Testing!
